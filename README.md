## Portfolio

The source code for my portfolio [website]("https://paulee.dev").

Build with [Vue.js](https://vuejs.org/), [TailwindCSS](https://tailwindcss.com/) and [Vite](https://vitejs.dev/).

### Project setup

```bash
git clone https://github.com/Paulanerus/paulee-site
cd paulee-site
npm install
npm run dev
```
